About the Project

This project was a gene sequencing tool written in Rust. It was intended to process read data from an Oxford Nanopore device, matching the reads onto a reference genome. It used a suffix tree for pattern matching, constructed using Ukkonen's algorithm. The idea was to exact match part of a read onto the genome and then check the approximate match of the read using edit distance. However, this turned out to be too unreliable, and a working approach would probably require approximate matching built into the suffix tree pattern matching. The project ended before we got to that. The team for this project had four members. I did research early on which determined the approach of the project. I wrote the suffix tree constructor, file IO, and pattern matching. Teammates wrote the edit distance function using dynamic programming and a size-optimized vec type to save space in the enourmous suffix tree created to represent the genome.

Technologies Used

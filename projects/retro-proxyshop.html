Retro Proxyshop Plugin

About the Project

Proxyshop is a Photoshop automation tool for rendering Magic: the Gathering cards. The Retro plugin adds support for old border cards. It consists of Python code and Photoshop template files. I patched together some of the textures from card scans and official renders, made the Photoshop templates, and programmed the plugin. I made this because I wasn't satisfied with the existing support for old border cards. I'm very happy that other people in the community use my plugin.

Technologies Used

  • Python
  • Adobe Photoshop
  • Image Processing
  • Windows COM Automation

My Projects

Retro Proxyshop Plugin

Proxyshop is a Photoshop automation tool for rendering Magic: the Gathering cards. The Retro plugin adds support for old border cards. It consists of Python code and Photoshop "template" files. Most of the textures were patched together, extended, and upscaled by x1uo3yd. I improved some of the textures, made the Photoshop templates, and programmed the plugin. This is a hobby project I made because I wanted it to exist.

Space Escape VR

This was my last big school project. It's a VR escape room game with a sci-fi setting. I led the project and coordinated a team of 8 students. The coding was done with UE5 blueprints. I did most of the coding. Other members made most of the 3D models and textures.

Bioinformatics

An implementation of gene sequencing by read alignment in Rust, using Ukkonen's algorithm for suffix tree construction. Approximate matching utilizied a dynamic programming solution for edit distance. I did research early on which was pivotal for deciding the direction of the project. I wrote the suffix tree constructor, file IO, and top level logic. Teammates wrote the edit distance function and a size-optimized vec variant to save memory in the staggeringly large suffix tree.
